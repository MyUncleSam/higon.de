# higon.de - High Graphics ON

A retro gaming-themed website featuring animated classic arcade games as an interactive background.

## Features

- **9 Classic Game Animations**: Pac-Man, Space Invaders, Asteroids, Galaga, Frogger, Centipede, Q*bert, Donkey Kong, and Super Mario Land
- **Auto-Rotating Display**: Games automatically switch every 5 seconds
- **Manual Selection**: Press keys 1-9 or click on the game list to select a specific game
- **Responsive Design**: Scales to any screen size

## Usage

Simply open `index.html` in a modern web browser. No installation or build process required.

### Keyboard Controls

| Key | Game |
|-----|------|
| 1 | Pac-Man |
| 2 | Space Invaders |
| 3 | Asteroids |
| 4 | Galaga |
| 5 | Frogger |
| 6 | Centipede |
| 7 | Q*bert |
| 8 | Donkey Kong |
| 9 | Mario Land |

After manual selection, auto-rotation pauses for 15 seconds.

## Technical Details

- Pure vanilla JavaScript (no frameworks or dependencies)
- HTML5 Canvas rendering
- All games are AI-driven demonstrations
